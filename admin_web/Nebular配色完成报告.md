### 阴影系统

**Tabler 风格**：
```css
box-shadow: 0 2px 8px 0 rgba(45, 62, 80, 0.12);
```

**Nebular 风格**：
```css
box-shadow: 0 2px 8px 0 rgba(45, 62, 80, 0.12);
悬停时：0 4px 12px 0 rgba(51, 102, 255, 0.35);
```

## ✨ Nebular 特色效果

### 1. 按钮光晕效果
悬停时产生品牌色光晕：
- Primary: 蓝色光晕 (`rgba(51, 102, 255, 0.35)`)
- Success: 绿色光晕 (`rgba(0, 214, 143, 0.35)`)
- Danger: 红色光晕 (`rgba(255, 61, 113, 0.35)`)

### 2. 卡片阴影
- 默认：柔和的灰色阴影
- 悬停：品牌色光晕 + 上移效果

### 3. 配色特点
- **更柔和的蓝色** (`#3366ff`)
- **更鲜活的绿色** (`#00d68f`)
- **更温暖的橙色** (`#ffaa00`)
- **更柔和的红色** (`#ff3d71`)
