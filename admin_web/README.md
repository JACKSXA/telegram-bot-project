# 📊 Bot管理后台

## 🚀 快速启动

### 方法1：使用启动脚本
```bash
cd /Users/hack/AI招聘/admin_web
./start.sh
```

### 方法2：手动启动
```bash
cd /Users/hack/AI招聘/admin_web
source ../venv/bin/activate
pip install -r requirements.txt
python flask_app.py
```

---

## 🌐 访问后台

启动后，在浏览器打开：
```
http://127.0.0.1:5000
```

**默认账号**：
- 用户名：`admin`
- 密码：`admin123`

---

## 📋 功能说明

### 1. 用户管理
- 查看所有用户列表
- 用户详情（状态、对话历史等）
- 搜索和筛选

### 2. 消息推送
- 批量推送消息
- 按状态筛选（全部用户/等待客服的用户）
- 支持变量：{username}, {wallet}

### 3. 数据统计
- 总用户数
- 用户状态分布
- 语言分布
- 活跃度统计

## 💡 使用示例

### 推送消息给所有用户
1. 进入"消息推送"
2. 选择"全部用户"
3. 输入消息内容
4. 点击"立即推送"

### 推送给等待客服的用户
1. 选择"等待客服的用户"
2. 输入个性化消息
3. 推送

---

## 📝 注意事项

- 后台需要Bot正常运行才能查看数据
- 推送消息会直接发送到用户Telegram
- 建议先测试再批量推送
