### 2. Web服务状态

### 3. Web服务运行时日志

**非常重要！**

点击 **Web服务** → **"Logs"** 标签（不是Build Logs！）

**运行时日志应该显示**：
```
Starting Flask application...
Running on http://0.0.0.0:PORT
```

**或者错误信息**：
```
ModuleNotFoundError: No module named 'flask'
ImportError: cannot import name 'app' from 'flask_app'
```
