## ✨ 动画效果

### 卡片悬停
```css
.card:hover {
    transform: translateY(-2px);  // 上浮2px
    box-shadow: 0 4px 20px rgba(0,0,0,0.15);  // 阴影增强
}
```

### 侧边栏选中
```css
.nav-link.active {
    background: linear-gradient(135deg, #667eea, #764ba2);
    box-shadow: 0 4px 15px rgba(102,126,234,0.4);  // 发光效果
}
```

### 漏斗步骤
```css
.funnel-step:hover {
    transform: translateX(5px);  // 右移5px
    box-shadow: 0 4px 20px rgba(0,0,0,0.15);  // 阴影增强
}
```
